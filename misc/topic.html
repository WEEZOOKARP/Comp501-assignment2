Focus in sub-topic: Security opportunities and risks of open source software

What is open-source software?

A program or set of files that fall under the open-source license. This means the program or files can be shared, edited, or used without the need for permission from the author of the program/software. This is permitted through an open-source license.

Why was this topic chosen?

  • Controversy: debates over open-source software’s security compared to proprietary applications’
  • Pros and cons: transparency, community-driven effort, vulnerabilities exposure, etc.
  • Comparative analysis: differences in security practices vs proprietary software’s
